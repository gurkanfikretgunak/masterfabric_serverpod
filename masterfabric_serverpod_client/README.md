# masterfabric_serverpod_client

This is the Serverpod client package for MasterFabric. The code in this package is mostly generated by Serverpod.

## Overview

This client package provides:

- Protocol definitions for communication with the server
- Generated endpoint clients
- App configuration models
- SerializableExceptions (RateLimitException, etc.)
- Notification models and stream handling

## Usage

```dart
import 'package:masterfabric_serverpod_client/masterfabric_serverpod_client.dart';

// Initialize client
final client = Client('http://localhost:8080/');

// Call endpoints
final response = await client.greeting.hello('World');
```

## Regenerating Code

When you modify models or endpoints in the server, regenerate the client code:

```bash
cd ../masterfabric_serverpod_server
serverpod generate
```
